Reflection

What went well:

This was the first project I created, so there was a bit of a learning curve in figuring out standard UI practices for mobile devices and how I wanted the design to look. I believe designs can always be improved, so I think it was good for me to come back and redesign this project to see how my deisgn skills have improved from the initial wireframes to the final high fidelity screens. Based on feedback I've gotten from my mentor and others, this project has become one of my stronger pieces through the re-design.

What could be improved:

This project was mainly learning about how to use my design tool, so I want to conduct more UX research and user testing to further develop my project and gain insight into how to impove it. I also want to design more screens to showcase additional funcitonalitites of the app.
